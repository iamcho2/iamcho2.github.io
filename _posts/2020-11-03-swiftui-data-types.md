# 자료형

### Multi-line strings

* Multi-line strings는 무조건 """ 쌍따옴표 3개가 앞뒤에 있어야 함. 또한 """는 한 줄에 혼자 있어야 함 ( abcd """ 이렇게 끝나면 안 됨)

  ```swift
  var tennyson = """
  Tis better to have loved
  and lost than never
   to have loved at all
  """
  ```
